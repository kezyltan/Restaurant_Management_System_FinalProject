# Web-Based Restaurant Inventory System

## Project Overview
A web-based inventory management system for restaurants built with PHP and SQLite.

## Requirements
- PHP 7.4 or higher
- SQLite (via PDO)

## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run using PHP dev server:
   php -S localhost:8000
4. Open browser and go to http://localhost:8000
