# Restaurant Inventory Stock Monitoring System

## Description
A Windows Forms application for monitoring restaurant inventory and stock levels.

## Features
- Add, edit, and delete inventory items
- Track stock in and stock out movements
- Low stock alerts
- Export inventory report to CSV

## Requirements
- Visual Studio 2019 or later
- .NET Framework 4.7+
- SQL Server LocalDB

## How to Run
1. Open the .sln file in Visual Studio
2. Build the solution
3. Run the application
